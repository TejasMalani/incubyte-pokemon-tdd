# incubyte-pokemon-tdd
TDD-based React application for Pokémon listing and detail view using PokeAPI, focusing on clean architecture, testing, and production-ready practices.

## 🚀 Overview
This project is a Pokémon listing and detail application built using React.

## 🧪 Approach
This project follows strict Test-Driven Development (TDD):
- Write failing tests
- Implement minimal code
- Refactor

## 🛠 Tech Stack
- React (Vite + TypeScript)
- React Router
- React Query
- Jest + React Testing Library
- Tailwind CSS

## 📦 Status
Project initialized. Further commits will follow TDD workflow.
